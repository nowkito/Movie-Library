# 🎬 Movie Library — Личная кинотека

**Автор:** Старостин Кирилл Валерьевич 

## Описание программы

Графическое приложение для ведения личной коллекции фильмов. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Данные сохраняются локально в файл `movies.json`. Доступна удобная фильтрация по жанру и году выпуска.
*Технологии:* Python 3.x и стандартная библиотека Tkinter.

## Инструкция по запуску

**Требования:**
Установленный интерпретатор Python версии 3.6 или новее.
Библиотека Tkinter входит в стандартную поставку Python.

**Шаги:**
1. Убедитесь, что Python установлен.
2. Скачайте или склонируйте этот репозиторий.
3. Запустите скрипт из корня проекта:
   ```bash
   python main.py   # Для Windows / Linux / macOS (в виртуальном окружении)
   ```
4. Используйте интерфейс программы для добавления и поиска фильмов.
5. Данные сохраняются автоматически при добавлении нового фильма.
6. Для сброса всей базы данных удалите файл `movies.json`.
7. Для отмены фильтрации просто перезапустите приложение или вручную очистите фильтры.
